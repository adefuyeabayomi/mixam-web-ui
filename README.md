## Description
Mixam is a global community platform for entrepreneurs to network, share insights, access free resources, and grow their businesses.  
This frontend is built with **Vue.js**, providing a responsive, interactive, and user-friendly interface for all Mixam users.

## Features
- User Authentication: Login / Signup
- Interactive Menu and Navigation
- Community access and posts
- Access to free courses and articles
- Event listings and networking opportunities
- Responsive design for desktop and mobile

## Tech Stack
- **Frontend Framework:** Vue.js
- **Styling:** CSS / SCSS
- **Routing:** Vue Router
- **State Management:** Vuex
- **API Calls:** Axios / Fetch API
